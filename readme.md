
## CELL

Single computation unit for simple I/O and (temporary) storage


## Install

Using npm: 

```
npm install cell
```


## Showcase 

Notable applications of cell:

* [Construct Cache](http://github.com/constructjs/cache)
* [PN:OI](http://pnoi.net)
